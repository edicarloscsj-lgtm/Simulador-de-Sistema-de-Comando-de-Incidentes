Como Usar o Sistema

O Instrutor

Inicia a sala de instrução, cria ameaças (fogo, fumo, químicos) no mapa, dita as condições meteorológicas e introduz injeções críticas de ocorrências no ecrã dos alunos via rádio.

A Equipa (Alunos)

Entram com o código gerado pelo instrutor. Lêm a ordem de operação (Briefing), ativam a Grade Cartográfica para comunicação tática, desenham as zonas de perigo e distribuem as viaturas no mapa.

Sistema C4I focado no apoio interativo ao Sistema de Comando de Incidentes.
